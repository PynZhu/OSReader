# OSReader

**OSReader** is a versatile, cross-platform e-reader built on the HarmonyOS OpenHarmony ecosystem. Designed for speed, privacy, and extensive format support, it aims to provide a seamless reading experience for learners and book enthusiasts.

**OSReader** 是一款基于鸿蒙 OpenHarmony 生态开发的全能电子书阅读器。专为高效阅读、隐私保护及多格式支持而生，旨在为学习者和阅读爱好者提供丝滑的阅读体验。

---

## ✨ Features | 功能特性

* **Multi-format Support**: Native support for PDF, TXT, EPUB, and MOBI files.
    **多格式支持**：原生支持 PDF, TXT, EPUB, MOBI 等多种电子书格式。
* **Intelligent Encoding**: Auto-detects GBK and UTF-8 encoding to eliminate garbled text in TXT files.
    **智能编码识别**：自动识别 GBK 与 UTF-8 编码，彻底告别 TXT 乱码困扰。
* **Performance First**: High-performance rendering engine capable of handling large documents with zero lag.
    **性能极致**：高性能渲染引擎，大文件加载不卡顿。
* **System Integration**: Deeply integrated with HarmonyOS file picker and sandbox security.
    **系统深度集成**：深度适配鸿蒙文件选择器与沙盒安全机制。

## 🛠️ Tech Stack | 技术栈

* **Language**: ArkTS / TS
* **Framework**: ArkUI / HarmonyOS SDK (API 12+)
* **Architecture**: Modular design (Entry & Core separation)
* **License**: GNU GPLv3

## 🚀 Getting Started | 快速开始

1.  Clone the repository:
    `git clone https://github.com/PynZhu/OSReader.git`
2.  Open the project in **DevEco Studio**.
3.  Sync dependencies via `ohpm install`.
4.  Run on your HarmonyOS device or simulator.
